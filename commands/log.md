# worklog:log — 写日报

智能汇总今日工作，写入日记。

## 配置

```
VAULT_PATH = "/Users/liutianpeng/projects/worklog"
```

## 流程

当此命令被调用时：

### 1. 获取今日 git 提交

在当前目录执行：
```bash
git log --since="midnight" --oneline --no-merges
```

如果不在 git 目录或执行失败，跳过此步，直接进入第 2 步。

### 2. 展示提交列表，询问补充

如果有提交，展示：
```
今天 git 提交（N 条）：
- {commit_hash} {commit_message}
- ...

今天主要做了什么？（补充说明、遇到的坑、学到的东西，或直接回车跳过）
```

如果没有提交：
```
今天主要做了什么？（描述你的工作内容）
```

### 3. 写入日记

- 文件路径：`VAULT_PATH/日记/YYYY-MM-DD.md`（YYYY-MM-DD 为今天日期）
- 如果文件不存在，创建新日记使用模板：
```markdown
# YYYY-MM-DD

## 今天要做的
- [ ]

## 工作记录

## 笔记

## 明天继续
```

- 追加到 `## 工作记录` 部分，格式：
```markdown
- {时段}：{综合描述}
```
  - 时段推断：如果现在是上午（12点前），写"上午"；否则写"下午"
  - 内容综合：git 提交要点 + 用户补充说明

### 4. 报告结果

```
已写入 日记/YYYY-MM-DD.md
```
